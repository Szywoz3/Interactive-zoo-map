Repozytorium zawiera interaktywną mapę poznańskiego zoo zrealizowaną w formie aplikacji webowej. Projekt został wykonany z wykorzystaniem technologii HTML5 oraz JavaScript i umożliwia użytkownikowi intuicyjną eksplorację mapy zoo poprzez interaktywne elementy.

Główny plik HTML "inz" odpowiada za strukturę aplikacji, natomiast logika interakcji, obsługa zdarzeń oraz funkcjonalność mapy zostały zaimplementowane w języku JavaScript plik "map". 
Pozostałe pliki w repozytorium (arkusze stylów, skrypty pomocnicze oraz zasoby) są niezbędne do zapewnienia pełnej i poprawnej funkcjonalności strony.

Projekt ma charakter edukacyjny i demonstracyjny, prezentując zastosowanie technologii frontendowych do tworzenia interaktywnych aplikacji internetowych.

This repository contains an interactive poznan zoo map implemented as a web application. The project is developed using HTML5 and JavaScript, allowing users to intuitively explore the zoo through interactive map elements.

The main HTML file "inz" defines the application structure, while the interaction logic, event handling, and map functionality are implemented in JavaScript "map" file. 
Additional files in the repository (stylesheets, supporting scripts, and assets) are required to ensure full and proper operation of the website.

The project is educational and demonstrative in nature, showcasing the use of frontend web technologies for building interactive web applications.
